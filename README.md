# rollup-plugin-glsl-optimize

Import GLSL source files, optimized with [Khronos Group SPIRV-Tools](https://github.com/KhronosGroup/SPIRV-Tools).

```js
import frag from './shaders/myShader.frag';
console.log(frag);
```
